# Robobuddy

Play with a friendly robo-buddy of your own in WebVR!

## Contributing

### Install

```bash
yarn install
```

### Start dev server

```bash
yarn start
```

### Create New Component

```bash
cd src/components
yarn angle initlocalcomponent component-name
```

### Format Code

```bash
yarn format
```

## Deploy

This app is auto-deploying on [Render](render.com) on every push to `master`.
